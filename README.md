# XChat

## 简介

XChat 是一个简单的，高自定义性聊天软件，支持在 Linux、Windows、Android 等任何一个安装了 Python3 的设备上运行。

## 文件结构

XChat<br>
├── client`客户端`<br>
│   └── cli`命令行版`<br>
│   │    └── main.py<br>
├── LICENSE`许可证`<br>
├── README.md`说明`<br>
└── server`服务端`<br>
│   └── server.py`服务端主程序`<br>

## 依赖

1. Python3
2. rich
3. PySide2（窗口客户端）

## 编译步骤

1、 克隆本仓库

```bash
git clone https://github.com/This-is-XiaoDeng/XChat.git
```

2、 安装 Python3

3、 安装 依赖库

```bash
pip install rich pyside2
```
